## picgo-plugin-itbaima-image

PicGo插件，添加栢码官网图床支持

---

plugin for [PicGo](https://github.com/Molunerfinn/PicGo)

Author: [FlapyPan](https://github.com/flapypan)

---

#### 注意事项

- 插件安装完后请在配置中添加 API token
- 最大可上传**3.0MB**的图片
- 插件暂时不支持同步删除图片

#### 安装和配置

直接在 picgo 插件设置里搜索 itbaima-image 安装
